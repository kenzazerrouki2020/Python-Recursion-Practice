🐍 **Python Recursion Practice – Removing Consecutive Duplicate Characters**

Today, I practiced one of the most interesting concepts in Python: **Recursion**.

The challenge was to write a recursive function that removes consecutive duplicate characters from a string.

**Example:**

```text
Input:
wwwwwwoooooorrrrrrrlllllllddddd

Output:
world
```

This exercise helped me understand:

* ✅ How recursive functions call themselves.
* ✅ The importance of a **base case** to stop recursion.
* ✅ How each recursive call processes a smaller part of the string.
* ✅ How results are combined while returning from the recursive calls.
